TRILATERATION - NodeMCU ESP 8266   -- DOUBLE-NODE
================================

This is the so called Double-Node. 
2 ESP8266 are combined into a DOUBLE...

1 Node is sniffing and the other Node is sending this Data to a Host ( e.g. IIs )

Key-infos:

2 Nodes
PJON to have a stable communication between those two Nodes
JSO for perfect dataFormat
MQTT for broadcasting and Synch
Communication to REST-Client

purpose of this DOUBLE-NODE is to scan the Beacon-Requests of Wifi-Mobile-Devices nearby

Detect: RSSI, MAC, SSID
